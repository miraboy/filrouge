Contexte du projet

Cette application de gestion d'entreprises a été développée dans le cadre du module de programmation web avancée.

Elle utilise une architecture MVC avec PHP objet et une base de données MySQL.

Fonctionnalités implémentées

Gestion des entreprises

  • Affichage de la liste complète des entreprises
  • Pagination des résultats (10 éléments par page)
  • Système de tri multi-critères (ID, Nom, Description, etc.)
  • Recherche globale dans toutes les données
  • Visualisation détaillée d'une entreprise
  • Ajout de nouvelles entreprises
  • Modification des informations existantes
  • Suppression d'entreprises avec confirmation

Fonctionnalités techniques

  • Architecture MVC bien structurée
  • Programmation orientée objet en PHP
  • Validation des formulaires côté serveur
  • Gestion des erreurs et messages utilisateur
  • Système de génération de données fictives
  • Interface responsive avec Bootstrap 5
  • Connexion sécurisée à la base de données
  • Upload et gestion des logos d'entreprises

Technologies utilisées

PHP 8+ MySQL Bootstrap 5 HTML5 CSS3 JavaScript jQuery DataTables

Remarques importantes

  • L'application a été testée sur Apache avec PHP 8.1
  • La base de données doit être configurée dans le fichier donnees.php
  • Toutes les actions sensibles (suppression, modification) sont confirmées
  • Le code source est commenté pour faciliter la maintenance
